# jobs/ — あなたの定型作業

夜間に実行する「定型作業」を、self-contained な job として置く場所。
各 job は新規セッションでも AI が迷わず実行できるよう、判断材料を全部ファイルに持つ。

> Phase 4 の `setup-job` スキルが、対話からこのディレクトリに job を生成する。
> ユーザーが手で書く必要はない。

## 想定レイアウト（Phase 1 以降で確定）

```
jobs/
  <job-name>/
    job.md        # 仕様 / 既存パターン参照 / 迷ったら / 禁止事項 / 完了条件 + enabled・schedule
```

`job.md` の5項目テンプレは「自律できる粒度までタスクを事前に固める」ゲート。
これが埋まらない作業は夜間に出さない。
